## 리스트

### 순서가 있는 리스트

1. 리스트가 만들어진다.`1.`
2. 엔터치면 자동으로 숫자가 만들어짐.
   1. `tab`누르면 세부사항 ㅇㅇ
3. `shift+tab`은 다시 밖으로
4. 완전히 나가고자 한다면 가장 밖에있는 리스트에서 한번더 들여쓰기
